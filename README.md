# Cats vs Dogs Image Classification using Transfer Learning

This project implements a high-performance image classification pipeline using
transfer learning with a pre-trained CNN (MobileNetV2) on the Kaggle Dogs vs Cats dataset.

> Framework: TensorFlow / Keras  
> Model: MobileNetV2  
> Task: Adapt a Pre-trained Model for Custom Image Classification

Detailed documentation will be added as the project progresses.
